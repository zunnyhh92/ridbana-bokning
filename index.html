GitHub Gist Konfiguration

För att lagra bokningar och inställningar behöver du skapa ett GitHub Gist.

  1. Gå till gist.github.com
  2. Skapa ett nytt gist med filnamnet booking-data.json
  3. Innehåll: {}
  4. Kopiera Gist ID från URL:en (efter gist.github.com/username/)
  5. Skapa en Personal Access Token på GitHub Settings
  6. Välj "gist" scope
